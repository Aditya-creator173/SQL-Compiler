## Installation

### Frontend (Node.js)
```bash
npm install
```

### Future Backend (Python - if needed)
This project is currently frontend-only. If you plan to add a Python backend for SQL execution:
```bash
pip install -r requirements.txt
```

## Tech Stack
- **Framework**: React 18 with Vite
- **Styling**: Tailwind CSS 3
- **Icons**: Lucide React
- **Build Tool**: Vite 5

## Development
```bash
npm run dev    # Start development server
npm run build  # Build for production
npm run preview # Preview production build
```
